    TUTORIAL (APP) LESSON

  • Learning Objective 1.2.1
    Students should be able to reproduce the functionality of the TUTORIAL APP on their own devices.
  • Learning Objective 5.5.1
    They should be able to use a number to define the radius of the paint dots.

    • Reflection entry in portfolio

  • Learning Objective 5.5.1
    Students should be able to describe all of the events used in the Paint Pot app (Click, Touch, and Drag).
  • Learning Objective 5.2.1
    In particular, they should mention that Touch events are brief and involve one location on a touchscreen, while Drag events involve sustained contact while moving a finger across the screen.
